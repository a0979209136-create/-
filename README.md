# 國中生物會考複習系統

這是新北市立義學國中生物會考複習系統原型，目標是協助七升八與八升九學生透過診斷、錯題筆記、復仇檢核、章節學習地圖、素養議題與排行榜進行系統化複習。

## 主要頁面

- `index.html`：學生端主網站與校內展示入口。
- `teacher-dashboard.html`：教師端學習監控與網站優化後台。
- `admin-dashboard.html`：教務端名冊、題庫、模考、權限與資料匯入匯出後台。
- `app.js`：學生端互動、診斷、圖像化錯題筆記、復仇檢核。
- `dashboard.js`：教師端 dashboard 示範資料與圖表渲染。
- `styles.css`：全站樣式。

## 資料範本

- `data/learning_logs.json`：學生學習紀錄範本。
- `data/feedback_inbox.json`：學生意見信箱範本。
- `data/student_directory.json`：學生名冊範本。

## 教師端新增功能

- 全班登入人數、今日使用人數、本週活躍學生。
- 未登入學生名單。
- 診斷題完成率、錯題筆記完成率、復仇檢核完成率。
- 各章節平均熟練度。
- 各概念錯誤率排行。
- 個別學生學習進度、弱點章節與補強任務。
- 模考前完成進度與八升九會考備戰進度。
- 學生個人報告。
- 班級與年級總覽。
- 學生意見信箱與教師端處理狀態。
- 網站優化雷達。

## 教務端新增功能

- 三層角色權限說明：學生端、教師端、教務端。
- 名冊、學習紀錄、意見信箱資料表規格。
- 題庫、模考週次、公告、教師權限與匯入匯出管理建議。
- 正式部署與個資風險提醒。

## 靜態版限制

目前此版本是靜態網站原型：

- 教師端 dashboard 使用假資料展示。
- 教務端後台是規格展示，不會真的寫入資料庫。
- 學生進度主要存在瀏覽器或示範資料中。
- 學生意見信箱目前是靜態範本，正式處理需要後端。
- 學生名冊不可硬寫在公開前端程式碼中。

## 正式上線建議後端

可選：

- Firebase。
- Supabase。
- Google Sheets + Apps Script。
- 學校伺服器。
- 教育局或校方 SSO / OAuth。

正式版需要後端支援：

- 學生登入與名冊驗證。
- 教師與教務權限控管。
- 學習歷程紀錄。
- 每週排行榜結算。
- 意見信箱處理。
- 題庫與時事資料更新。
- 資料匯出與備份。

## 權限風險提醒

學生端不可看到教師端與教務端資料。正式公開部署時，請務必：

- 將教師端與教務端放在受保護路由。
- 使用後端檢查登入者角色。
- 不在公開 JS 放真實學生名冊。
- 不公開完整學生姓名、學號、信箱。
- 題庫需確認授權，考古題建議連回官方頁面。
