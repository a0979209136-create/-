教務維護入口

目前版本為「靜態展示與教務維護說明版」。

本頁只說明題庫、章節、時事、公告與模考時程如何維護。暫時不做真實登入、真實名冊驗證或真實學生追蹤。

正式追蹤需接後端

若要追蹤學生學習狀況，需接 Google Sheet + Apps Script、Firebase、Supabase 或校內伺服器。

資料維護

目前公開資料都是示範資料

題庫資料

維護 data/questions.json。每題需包含章節、概念標籤、選項、答案與詳解。

章節資料

維護 data/chapters.json。每章保留常考重點、迷思、學習地圖與圖像記憶。

素養時事

維護 data/news_database.json。新增時請放可信新聞、官方資料或期刊來源連結。

重要公告

維護 data/announcements.json。只放學生需要知道的任務提醒。

模考時程

維護 data/mock_exam_schedule.json 與 data/exam_calendar.json。

測試稱號

維護 data/demo_student_directory.json。只可放測試代號與稱號，不放真實姓名或學號。

部署與更新

GitHub 更新後由 Netlify 重新部署

更新流程

修改網站檔案後推到 GitHub main。若 Netlify 未自動部署，請到 Netlify 後台按 Trigger deploy。

靜態限制

靜態網站可顯示題目、章節、公告與時事，但不能安全保存真實學生紀錄。

資安提醒

公開網站不放個資

不得公開

  • 真實學生姓名、真實學號、真實班級座號。
  • 真實登入驗證規則、教師權限資料。
  • 未授權考古題原文或圖片。
